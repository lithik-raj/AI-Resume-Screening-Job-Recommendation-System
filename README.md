# AI Resume Screening & Job Recommendation System

## Project Overview

The AI Resume Screening & Job Recommendation System is a Machine Learning-based web application that automates resume analysis, ATS score calculation, skill extraction, role prediction, and job recommendations.

The system helps job seekers evaluate their resumes while assisting recruiters in identifying suitable candidates efficiently. It uses Natural Language Processing (NLP) and Machine Learning techniques to analyze resume content and provide intelligent career recommendations.

---

## Features

* Resume Upload (PDF Format)
* Resume Text Extraction
* Skill Detection and Analysis
* ATS Score Calculation
* Machine Learning Based Role Prediction
* AI Job Recommendation System
* Interactive Dashboard
* Modern User Interface (UI/UX)
* Recruiter-Friendly Screening Process

## Project Workflow

1. User uploads a resume in PDF format.
2. Resume text is extracted using PDF processing.
3. Skills are identified from the resume.
4. ATS score is calculated.
5. Machine Learning model predicts the most suitable job role.
6. Relevant job recommendations are displayed.
7. Results are presented through an interactive dashboard.
